# AI Agent Execution

Source: https://app.notion.com/p/37ee3c20a8ee81a0b19cfa8d04546c08

Orchestration system for running automated AI agent workflows daily.

## Structure

- `collection-of-prompts/` — execution-trigger prompts for automated workflows
- `start-the-execution.md` — orchestrator: runs all prompts end-to-end in order
- `description.md` — plain-language reference for the full pipeline
- `ai-agents-memory.md` — shared memory layer (Notion databases) for the investment agent set
- `sprint.md` — sprint tracking
