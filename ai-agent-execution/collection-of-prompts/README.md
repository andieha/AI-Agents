# Collection of Prompts

Source: https://app.notion.com/p/37ee3c20a8ee81fd924cf389e19f9fa1

Groups the core execution-trigger prompts used to run automated workflows.

## Prompts

- **Start Collecting All** — runs News, Field Monitor, Email, and Daily Intelligence Brief collection in parallel.
- **Clean All** — applies retention rules across all report sections, archiving excess pages.
- **Daily Intelligence Brief — Sub-Agent Prompt** — the underlying prompt used by the Daily Intelligence Brief sub-agent.
- **Email Whitelist Monitor — Prompt** — monitors whitelisted senders for priority email handling.
- **Aggregated Report (TTS)** — merges today's Daily Brief reports into a clean, TTS-ready summary.
- **Collecting - Newsletter Digest (Expand)** — cleans up newsletter emails from Gmail and saves under AI Reports.
- **Collecting - AI/Robotics Investment Signals** — runs the 3-agent investment monitoring pipeline.
