# Field Monitor

**Trigger:** Called by `start.md` as part of Step 1 (parallel). Requires the Collection Drive folder ID from the caller.

7-day analysis on "AI Agents & Tool Use".

## Steps

1. Run 4+ web searches on AI Agents & Tool Use developments over the last 7 days.
2. Group findings into themed clusters.
3. Include "So What" section — implications and takeaways.
4. Include "Who to Follow" section — key people or sources to watch.

## Output

- Create a Google Doc **inside the Collection folder** (parent = Collection Drive folder ID).
- Title: `📡 Field Monitor Digest – Week [N], [YEAR]`
- All content in English.

- 5xx error → wait 10s, retry up to 3×, then skip and continue
